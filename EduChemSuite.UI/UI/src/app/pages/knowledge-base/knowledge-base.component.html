Getting Started

Logging In

  1. Navigate to your EduChemSuite instance URL.
  2. Enter your email and password.
  3. Click Log In.

If you don't have an account, ask your administrator for a registration invite link.

First-Time Setup

After your first login, complete your profile:

  1. Click Account > Profile in the sidebar.
  2. Click Edit to update your information.
  3. Fill in all required fields: address, city, state, country, zip code, and phone number.
  4. Click Save to apply your changes.

Navigating the Sidebar

The sidebar provides access to all features based on your role:

  • Welcome — Your dashboard with quick stats
  • Administration (Admin/Staff) — Districts, Schools, Users
  • Assessments — Exams and Grades
  • Content (Staff) — Questions, Image Types, Molecule Editor, Chemistry Tools
  • Messages — In-app messaging
  • Data (Staff) — CSV import/export
  • Account — Profile settings and logout
  • Knowledge Base — This documentation

Dark Mode

EduChemSuite supports dark mode:

  1. Toggle the sun/moon icon at the bottom of the sidebar, or
  2. Go to Account > Profile > Edit and toggle the Dark Mode switch.

Your preference is saved and applied automatically on future logins.

Email Privacy

By default, your email is hidden from other users in messaging. To change this:

  1. Go to Account > Profile > Edit.
  2. Toggle the Show Email switch.
  3. Click Save.

Editing & Versioning

When you edit an existing question, EduChemSuite automatically increments the version number. Previous versions are preserved so exam history stays accurate.

Soft Deletion

Deleting a question sets it as inactive rather than removing it permanently. Inactive questions no longer appear in searches but remain linked to existing exams for historical records.

Interactive Chemistry Tools

EduChemSuite provides specialized interactive tools for chemistry question types. When creating a question, selecting a chemistry question type will display the corresponding tool for building the correct answer.

Molecule Editor (Chemical Structure & Lewis Dot Structure)

The molecule editor (also accessible under Content > Molecule Editor) lets you build chemical structure diagrams:

  • Add atoms by selecting an element and clicking the canvas.
  • Create bonds by dragging between atoms. Supports single, double, and triple bonds.
  • For Lewis Dot Structures, add lone pairs and the tool calculates formal charges automatically.
  • The correct answer diagram is saved with the question for auto-grading.

Atomic Structure Builder (Bohr Model)

Students construct Bohr models of atoms:

  • Place protons and neutrons in the nucleus.
  • Place electrons in the correct electron shells (energy levels).
  • The tool validates shell capacity (2, 8, 18, 32 electrons per shell).

Chemical Equation Balancer

Teachers define an unbalanced chemical equation, and students balance it:

  • Set the reactants and products with their chemical formulas.
  • Students adjust coefficients to balance each element.
  • The tool verifies that all elements are balanced on both sides.

Electron Configuration Builder

Students fill in orbital diagrams for a given element:

  • Orbitals are listed in filling order (1s, 2s, 2p, 3s, etc.).
  • Students specify the number of electrons in each orbital.
  • Grading checks the total count and distribution against the correct configuration.

Periodic Table Quiz

An interactive periodic table where teachers can create questions about:

  • Element identification (name, symbol, atomic number).
  • Element classification (metals, nonmetals, metalloids).
  • Periodic trends (electronegativity, atomic radius, ionization energy).

Stoichiometry Stepper

A guided dimensional analysis tool for multi-step stoichiometry problems:

  • Students work through conversion factors step by step.
  • Each step includes a numerator and denominator with units.
  • The tool checks both the final numerical answer and the conversion path.

Building Exams

Creating an Exam

  1. Go to Assessments > Exams.
  2. Click Create Exam.
  3. Enter a name, description, and configure settings:
    • Time Limit — Maximum minutes allowed (optional).
    • Allow Retakes — Whether students can retake the exam.
    • Max Attempts — Number of allowed attempts if retakes are enabled.
  4. Click Save.

Adding Questions to an Exam

One at a time:

  1. Open an exam and click Add Question.
  2. Search for and select a question.

Batch add from Questions page:

  1. Go to Content > Questions.
  2. Use the checkboxes to select multiple questions.
  3. Click the floating "Add to Quiz" button.
  4. Choose to create a new exam or add to an existing exam.

For Students

Viewing Your Exams

  1. Go to Assessments > Exams in the sidebar.
  2. You'll see a list of exams assigned to you.

Taking an Exam

  1. Click on an exam to open it.
  2. Click Start Exam (or Resume if you previously started).
  3. Answer each question:
    • Multiple Choice / True-False: Select your answer from the options provided.
    • Essay: Type your response in the text area.
    • Chemical Structure / Lewis Dot Structure: Use the molecule editor to draw atoms, bonds, and (for Lewis structures) lone pairs.
    • Atomic Structure: Build a Bohr model by placing protons, neutrons, and electrons in the correct shells.
    • Chemical Equation: Adjust coefficients to balance the given equation.
    • Electron Configuration: Fill in the orbital diagram with the correct number of electrons per orbital.
    • Periodic Table Quiz: Use the interactive periodic table to answer questions about elements.
    • Stoichiometry: Work through a multi-step dimensional analysis problem using the guided stepper.
  4. Your responses are saved as you go.
  5. When finished, click Submit Exam.

Important Notes

  • If the exam has a time limit, a timer will be displayed. The exam auto-submits when time runs out.
  • If retakes are allowed, you can take the exam again up to the maximum number of attempts.
  • Once submitted, your exam is graded automatically in the background. You'll receive a notification when your grade is ready.

Viewing Your Grades

  1. Go to Assessments > Grades in the sidebar.
  2. You'll see a list of your graded exams with scores.

Reviewing Exam Results

After grading is complete:

  1. Click on a graded exam to see your results.
  2. For each question, you can see:
    • Your submitted answer
    • The correct answer
    • Whether your answer was correct

Messaging Your Teacher

If you have questions about an exam or grade:

  1. Go to Messages in the sidebar.
  2. Click Compose.
  3. Select your teacher from the recipient list.
  4. Write your message and send.

See Messaging for more details.

For Administrators

Managing Districts

Districts are the top-level organizational unit.

  1. Go to Administration > Districts.
  2. Click Create District to add a new one.
  3. Enter the district name and save.

Each district can contain multiple schools.

Managing Schools

  1. Go to Administration > Schools.
  2. Click Create School to add a new one.
  3. Enter the school name and assign it to a district.
  4. Save.

Managing Users

Viewing Users

  1. Go to Administration > Users.
  2. Use the search and filter tools to find users by name, email, district, school, or account type.

User Roles

RoleAccess Level AdminFull access to all districts, schools, and users. AdminStaffAccess to their assigned districts and schools. Can manage users within their scope. StaffAccess to their assigned schools. Can create content and manage students in their schools. StudentCan view assigned exams, take assessments, and see their grades.

Data Scoping

Users only see data within their scope:

  • Admins see everything.
  • AdminStaff see users and data in their districts and schools.
  • Staff see users and data in their schools.
  • Students see only their own data and can message staff in their school.

Registration & Invitations

New users are invited via email-based registration tokens:

  1. Generate a registration invite for a user.
  2. The user receives a link to create their account.
  3. Once registered, assign them to the appropriate district(s) and school(s).

Assigning Users to Districts & Schools

Users can belong to multiple districts and schools:

  1. Open a user's profile from Administration > Users.
  2. Assign them to districts and/or schools as needed.
  3. Their access scope updates automatically based on their role and assignments.

Bulk Operations

For managing large numbers of users, districts, or schools, use the CSV import feature. See Data Management for details.

Data Management

The Data Management page (under Data in the sidebar) provides CSV import and export tools for bulk operations. This feature is available to Staff and Admin users.

  1. Go to Data in the sidebar.
  2. Select the Export tab.
  3. Choose the entity type to export (Users, Districts, Schools, Questions, Exams, Grades, or Exam Responses).
  4. Optionally use search filters to narrow the results.
  5. Click Export to download a CSV file.

Importing Data

How to Import

  1. Go to Data in the sidebar.
  2. Select the Import tab.
  3. Choose the entity type you want to import.
  4. Download the CSV template for that entity to see the expected format.
  5. Fill in the template with your data.
  6. Upload the CSV file and click Import.

Import Behavior (Upsert)

Imports use upsert logic — matching records are updated, new records are created:

EntityMatch Criteria DistrictsDistrict name (case-insensitive) SchoolsSchool name + district name (auto-creates district if missing) UsersEmail (case-insensitive; password required for new users) QuestionsQuestion text + author email (increments version on update) AnswersQuestion text + answer text GradesStudent email + exam name

Import Results

After an import completes, you'll see a summary:

  • Total rows processed
  • Created count (new records)
  • Updated count (existing records modified)
  • Errors with row-level details

Tips

  • Always download and use the template to ensure correct column headers.
  • For user imports, include a password column for new users.
  • School imports will automatically create districts that don't exist yet.
  • Question imports increment the version number when updating existing questions.

FAQ

Account & Login

I forgot my password. How do I reset it?

Click the Forgot Password link on the login page and follow the instructions. If that doesn't work, contact your administrator.

How do I get an account?

Accounts are created through registration invitations. Ask your school administrator for an invite link.

How do I change my password?

Go to Account > Profile, click Edit, and update your password fields.

Exams & Grading

My exam grade shows as "pending." What does that mean?

Exams are graded automatically in the background after submission. If your grade is still pending, the grading process is in progress or some diagram questions need manual review by your teacher.

Can I retake an exam?

Only if your teacher has enabled retakes for that exam. You'll see a Retake option if allowed, up to the maximum number of attempts.

My answer was marked "pending review." Why?

Some chemistry tool questions (Chemical Structure, Atomic Structure, Lewis Dot Structure) can flag partially correct answers for your teacher to review manually. For example, if your molecule has the right atoms and bonds but the angles are slightly off, or your Lewis structure has correct topology but wrong lone pairs.

I submitted my exam but don't see my grade yet.

Grading runs in the background. You'll receive a notification when it's complete. This usually takes just a few seconds.

Questions & Content

How do I create a chemistry tool question?

When creating a question, select the desired chemistry question type (e.g., Chemical Structure, Atomic Structure, Chemical Equation, etc.). The corresponding interactive tool will appear so you can build the correct answer. Students will see the same tool when taking the exam.

What happens when I edit a question that's already on an exam?

Editing a question creates a new version. Existing exams keep the version of the question that was on them when students took the exam.

What chemistry tools are available?

EduChemSuite includes 7 interactive chemistry tools: Chemical Structure (molecule editor), Atomic Structure (Bohr model), Chemical Equation (balancer), Electron Configuration (orbital filling), Lewis Dot Structure, Periodic Table Quiz, and Stoichiometry (dimensional analysis stepper). See For Teachers for details on each tool.

Data Management

What format should my CSV file be in?

Download the template for the entity type you want to import. Use that as your starting point to ensure correct column headers and formatting.

Will importing overwrite existing data?

Imports use upsert logic — records matching the criteria (e.g., same email for users) are updated, while new records are created. See Data Management for match criteria details.

Can I undo an import?

There's no automatic undo. We recommend exporting your current data before importing to have a backup.

Messaging

I can't find a user to message. Why?

You can only message users within your data scope. Students can only message staff in their school. Staff can message users in their school. Admins can message anyone. See Messaging for details.

Are messages delivered in real time?

Yes. Messages are delivered instantly via SignalR. You don't need to refresh the page.

General

How do I switch to dark mode?

Click the sun/moon icon at the bottom of the sidebar, or toggle it in your profile settings.

The page isn't loading correctly. What should I do?

Try these steps:

  1. Refresh the page (Ctrl+F5 for a hard refresh).
  2. Clear your browser cache.
  3. Try a different browser.
  4. If the problem persists, contact your administrator.
